# 学生成绩管理系统

基于 Flask + SQLite + HTML/JavaScript(Ajax) 的 Web 成绩管理系统。

## 功能

- 用户登录 / 注册
- 学生信息管理（增删改查）
- 课程信息管理（增删改查）
- 成绩录入与管理
- 关键词搜索
- 成绩统计图表

## 运行

```bash
pip install -r requirements.txt
python app.py
```

浏览器访问：http://127.0.0.1:5000

默认账号：`admin` / `123456`

## 技术栈

- 后端：Python Flask
- 数据库：SQLite
- 前端：HTML + CSS + JavaScript（fetch Ajax）
